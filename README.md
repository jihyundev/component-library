## Component Library

Personal component playground. All component designs are based on [Material UI Community version](https://www.figma.com/community/file/912837788133317724).  

## How to run
```bash
  # use node v18.16.1
  nvm use 18.16.1

  # install packages
  yarn
  
  # run application
  yarn dev
  
  # run storybook
  yarn storybook
```
